Travel Planner

Github Repo | Github Live

First Major Group Project accomplished within 8 days of code work.

Technologies: HTML, CSS, JavaScript, jQUery, AJAX, and Firebase

Description: The main feature of our app is to build a list for travellers and locals to utilize for their trip. Date and destination are pre-selected to pull data from the Weather API to assist the user in planning their activities for the day. User gets to do a broad search for an activity (e.g. museum) or specific search of their point of interest (e.g. ROM). With the Google API we have selected, users can also view the images and reviews for the event. Final output is to compile these lists for users to refer to as they depart.

Crystal Bonanza

Github Repo | Github Live

Technologies Used: HTML, CSS, JS, jQuery

Description: A one-player game where the objective is to match the final score to the random score that was generated. Each crystal on the page has a random number attached to it so the user has to figure out what the value of each crystal is to help them get to the score they desire to attain. If the score matches, the user gets a winning point and if the user exceeds the random number then the user gets a losing point.
